## HTML Tag

---

1. Head

   - ```
     <h1> ~ </h1>
     <h2> ~ </h2>
     ~
     <h6> ~ </h6>
     ```

   - <h2>Head</h2>

2. Underline

   - ```
     <u> ~ </u>
     ```

   - <u>Underline</u>

3. Strong

   - ```
     <Strong> ~ </Strong>
     ```

   - <Strong>Strong</Strong>

4. 줄바꿈

   - ```
     <br>
     ```

   - 줄<br>바꿈

5. 단락

   - ```
     <P>
     ```

   - 단<p>락

6. Image

   - ```
     <img src="~">
     ```

   - <img src="파일이름 or url">

7. List

   - ```
     <li> ~ </li>
     ```

   - <li>List</li>

8. List 경계

   - ```
     <ul>
     ```

     - 

       <ol> 
           <li>List1</li>
           <li>List2</li>
           <li>List3</li>
       </ol>

     - 

   - ```html
     <ul>
         <li>List1</li>
     </ul>
     <ul>
         <li>List2</li>
     </ul>
     ```

     

   - List1<ul>List2

9. List Numbering

   - ```
     <ol> ~ </ol>
     ```

   - 

     <ol> 
         <li>List1</li>
         <li>List2</li>
         <li>List3</li>
     </ol>
